Sobre o Achei Cão – Local / Como usar

Esta página foi criada como um projeto para facilitar o reencontro de cães perdidos na região de Araucária. No futuro, pode ser usada como ferramenta real pela comunidade.

1. Como cadastrar um cão

  • Clique em “Cadastrar” ou em “Cadastrar um cão”.
  • Escolha se o cão está perdido ou encontrado.
  • Preencha bairro, porte, cor e, se possível, uma foto.
  • Informe um contato válido (telefone, WhatsApp ou e-mail).
  • Crie uma senha para depois poder remover o cadastro.

2. Como buscar e usar os filtros

  • Use os filtros de bairro, porte, raça e cor.
  • Clique em “Buscar” para atualizar a lista e o mapa.
  • Ao encontrar alguma correspondência, use o contato informado para falar com o responsável.

3. Boas práticas e testes com usuários

  • Evite colocar dados muito sensíveis no campo de contato.
  • Teste o site com donos de cães e colegas, observando se entendem o fluxo.
  • Use o feedback para ajustar filtros, texto e visual.
